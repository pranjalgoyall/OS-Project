🔹 🚀 Scalable Thread Management Simulator | Operating Systems Project

Turned abstract operating system concepts into an interactive, real-time experience by designing and developing a web-based scalable thread management simulator using HTML, CSS, and JavaScript. This project focuses on visualizing how modern systems handle concurrency at scale—bringing thread behavior out of theory and into motion.

🧵 Simulates High Concurrency at Scale
Engineered the system to handle 1,000+ concurrent threads, each with dynamic state transitions (New → Ready → Running → Waiting → Terminated), enabling realistic modeling of multi-threaded environments.

⚙️ Real-Time Scheduling & Synchronization
Implemented core OS concepts including thread lifecycle management, scheduling strategies (e.g., Round Robin / FCFS), and synchronization mechanisms (locks/mutex simulation) to demonstrate how threads compete and coordinate for resources.

📊 Performance Optimization & Smooth Rendering
Optimized event handling and DOM updates to achieve ~60 FPS smooth performance, reducing UI lag by approximately 35–40%, ensuring responsive and seamless visualization even under high load.

🎮 Interactive & Intuitive Visualization
Designed a user-friendly interface that allows users to observe thread execution, scheduling decisions, and synchronization effects in real time—transforming complex concurrency concepts into an intuitive learning experience.

🧠 Bridging Theory with Practical Understanding
This project strengthened my understanding of Operating Systems, concurrency, and performance optimization, while demonstrating how low-level concepts can be effectively simulated using modern web technologies.
